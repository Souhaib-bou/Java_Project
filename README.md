# Hirely – Job Application Management Platform

## Overview
This project was developed as part of the PIDEV – 3rd Year Engineering Program at Esprit School of Engineering (Academic Year 2025–2026).

Hirely is a platform that helps users manage job applications, track statuses, and organize recruitment processes.

## Features
- Job offer/application management
- interview management
- forum management
- onboarding plan/task 
- User management

## Tech Stack

### Frontend
- JavaFX

### Backend
- Java
- Spring Boot
- MySQL

## Architecture
MVC architecture with REST services between frontend and backend.
